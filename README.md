## 🖥️ Como visualizar o projeto localmente

1. Clone este repositório:
   ```bash
   git clone https://github.com/karlakryz-codes/meu-portf-lio.git
   ```
2. Acesse a pasta:
   ```bash
   cd meu-portf-lio
   ```
3. Abra o arquivo `index.html` no seu navegador.
